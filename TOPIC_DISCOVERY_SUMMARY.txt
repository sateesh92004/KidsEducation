PROBLEM STATEMENT
================================================================================

COMPLAINT:
  "Question papers were generated successfully, but when I tried to choose
   that topic, it's not listed under the topics list!"

SYMPTOMS:
  ❌ Papers are generated successfully
  ❌ Papers are saved to disk
  ❌ But student dashboard only shows hardcoded placeholder topics
  ❌ Real generated topics never appear in dropdown
  ❌ Students cannot access generated papers

ROOT CAUSE:
  The `populate_topics()` method used hardcoded placeholder topics instead
  of discovering actual topics from saved papers. It was a TODO that was
  never implemented.

================================================================================
SOLUTION
================================================================================

PART 1: ADD TOPIC DISCOVERY METHOD

FILE: app/services/question_service.py
METHOD: get_available_topics(grade: str, subject: str) -> List[str]

WHAT IT DOES:
  1. Scans app/data/ directory for paper files
  2. Extracts grade, subject, and topic from filenames
  3. Filters by matching grade and subject
  4. Returns sorted list of available topics
  5. Only includes topics with valid papers

EXAMPLE:
  Input: grade="10", subject="Math"
  ↓
  Finds files:
    - paper_10_Math_Algebra_Basics_p1.json
    - paper_10_Math_Algebra_Basics_p2.json
    - paper_10_Math_Geometry_p1.json
  ↓
  Output: ["Algebra Basics", "Geometry"]


PART 2: UPDATE STUDENT DASHBOARD

FILE: app/ui/student_dashboard.py
METHOD: populate_topics() (UPDATED)

BEFORE (BROKEN ❌):
  def populate_topics(self):
      placeholder_topics = [
          "Algebra", "Geometry", "Trigonometry", "Calculus",
          "Photosynthesis", "Cellular Biology", "Genetics", "Evolution"
      ]
      self.topic_combo.addItems(placeholder_topics[:4])

AFTER (FIXED ✅):
  def populate_topics(self):
      grade = self.grade_combo.currentText()
      subject = self.subject_combo.currentText()
      
      if not grade or not subject:
          return
      
      available_topics = self.question_service.get_available_topics(grade, subject)
      
      if available_topics:
          self.topic_combo.addItems(available_topics)
      else:
          self.topic_combo.addItem("No topics available")
