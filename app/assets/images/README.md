# Character Images for Login Screen

This directory should contain the character images displayed on the login screen.

## Required Images

1. **left_character.png** - The male character image (should be placed on the left side)
2. **right_character.png** - The female character image (should be placed on the right side)

## Image Specifications

- **Format**: PNG (recommended) or JPG
- **Recommended Size**: At least 700x1000 pixels (will be automatically scaled to 350x500)
- **Background**: Transparent or matching the login screen gradient

## How to Add Images

1. Save your character images in this directory
2. Name them exactly as:
   - `left_character.png` (for the left side)
   - `right_character.png` (for the right side)
3. The login screen will automatically load and display them

## Note

If the images are not found, the login screen will display a placeholder emoji (👤) instead.
